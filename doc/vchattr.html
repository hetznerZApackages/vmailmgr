NAME

vchattr - change a virtual user attribute

SYNOPSYS

vchattr [-c | --msgcount | -e | --expiry | -E | --enabled | -q | --softquota | -Q | --hardquota | -z | --msgsize] [--quiet] value username [username ...]

DESCRIPTION

This program changes the value of one attribute on a set of virtual users. It cannot be used to change the user's password or forwarding addresses -- use vpasswd and vchforwards to accomplish those tasks.

OPTIONS

-c count
--msgcount count

Limit the number of messages in this user's mailbox to count.

-e seconds
--expiry seconds

Set the account to expire at the given time, expressed in the number of seconds since the UNIX epoch (midnight GMT Jan 1, 1970).

-E 0|1
--enabled 0|1

Enable (1) or disable (0) delivery to the virtual user's mailbox directory. This does not delete the mailbox or any of the messages contained in or, nor prevent the user from logging in.

-q kbytes
--softquota kbytes

Set the soft quota value -- if the user's mailbox is this large or larger (but not larger than the hard quota), incoming messages will be both bounced and received. As well, an error message will be added to the user's mailbox.

-p "string"
--personal "string"

Set the user's personal information string.

-Q kbytes
--hardquota kbytes

Set the hard quota value -- if the user's mailbox is this large or larger, no more messages will be delivered to it.

-z kbytes
--msgsize kbytes

Set the largest message this user is allowed to receive.

RETURN VALUE

0 if the given attribute was successfully changed for all users, non-zero otherwise. If any of the steps fail, a diagnostic message is printed.

SEE ALSO

vadduser(1)

NOTES

This program expects the environment variable HOME to be set, and executes a change directory to the contents of it before starting. It is also required that you change user to the domain owner before using these utilities.
